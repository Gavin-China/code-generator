## 📢 项目简介

> 企业后端管理系统开发过程中，大量的重复增删改查 CRUD 的功能模块开发，通过标准化的代码自动生成器，四步配置即可实现规范的程序开发流程，有效提升开发效率、降低开发错误。

## 💪 功能特性

> - 后端基于 Spring Boot，前端基于 Vue3 、Element Plus。
> - 使用 Velocity 模板引擎进行代码生成。
> - 使用 Dynamic-datasource 进行动态数据源切换。
> - 灵活的数据字典，生成下拉框数据。
> - 全局提供暗黑视觉模式。

## 💻 运行环境

> 后端：JDK >= 1.8
>
> 前端：Node >= 12

## ✨ 操作说明

> 1.数据库建业务数据表，并对字段进行备注
>
> 2.启动项目前后端代码
>
> 3.配置链接数据库
>
> 4.导入数据表并选中
>
> 5.配置数据表信息及导出的相关信息
>
> 6.生成前后端代码
